## Overview

Anton K provides a centralized hub for your project documentation. You can create, organize, and publish docs using MDX, with built-in support for interactive components, version control, and search. Start building your knowledge base in minutes and scale as your project grows.

<Callout kind="info">
  Anton K integrates seamlessly with GitHub for versioned docs and supports custom domains for branded experiences.
</Callout>

## Key Features

Anton K streamlines documentation workflows with powerful tools tailored for developers and teams.

## Quick Start

Get your first documentation page live in under 5 minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://app.antonk.com` with your email or GitHub account.
  </Step>
  <Step title="Add Repository" icon="git-commit">
    Connect your GitHub repo containing MDX files.
  </Step>
  <Step title="Publish Docs" icon="rocket">
    Deploy your site with one click. Customize themes and add your brand color (`#3B82F6`).
  </Step>
</Steps>

Here's a sample MDX frontmatter for your pages:

<CodeGroup tabs="Default,Custom">
  ```yaml
---
title: My Page
description: Page description here.
---
  ```
  ```yaml
---
title: Advanced Page
description: Custom docs page.
brandColor: "#3B82F6"
---
  ```
</CodeGroup>
