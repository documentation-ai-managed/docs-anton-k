## Starting a Project

To create your first project in Anton K:

1. Log in to your Anton K dashboard
2. Click "Create Project" on the home screen
3. Enter a project name and description
4. Click "Save"

You can now add documentation pages, manage members, and organize content inside your new project.

For more options, explore the "Settings" tab on your project page.
